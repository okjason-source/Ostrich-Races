# PWA Icons

This directory contains the icon files for the Progressive Web App, generated from `ostrich-bird-shape-running.png`.

Required icon files:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Generating Icons

### Option 1: Browser-Based (Recommended)
1. Open `generate-icons.html` in your browser
2. The icons will be generated automatically from `ostrich-bird-shape-running.png`
3. Click any icon to download, or use "Download All Icons" button
4. Save all icons to this `icons/` directory

### Option 2: Command Line (Node.js)
1. Install dependencies: `npm install canvas` (optional)
2. Run: `npm run generate-icons` or `node generate-icons.js`
3. Icons will be automatically saved to this directory

## Icon Design

The icons feature:
- The running ostrich silhouette from `ostrich-bird-shape-running.png`
- Opulent gradient background with bright, clashing colors:
  - Gold (#FFD700)
  - Neon Pink (#FF00FF)
  - Electric Blue (#00FFFF)
  - Hot Pink (#FF1493)
- Properly scaled and centered for each size
- Matches the "Billionaire Mindset" theme aesthetic
